### Prompt #1 — Top 10 Customers by Revenue
```text
Act as a Senior SQL Engineer with expertise in PostgreSQL, MySQL, SQL Server, Oracle, and ANSI SQL.

Generate an optimized SQL query to retrieve the top 10 customers by total revenue generated during the last 12 months.

Requirements:
- Use ANSI SQL whenever possible
- Handle null values safely
- Include customer_id
- Include customer_name
- Include total_revenue
- Sort by highest revenue first
- Exclude canceled orders
- Support datasets larger than 100 million rows

Also provide:
1. Query explanation
2. Recommended indexes
3. Expected execution plan behavior
4. Performance optimization recommendations
5. Alternative query versions for PostgreSQL and SQL Server
```
